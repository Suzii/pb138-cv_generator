Záverečná správa

Mojou úlohou na projekte bolo vytvoriť transformaciu z XML do Jsonu, z Jsonu do XML a XML Schému.

Json -> XML

Transformacia Json do Xml sa poživa na transformaciu json dát vygenerovaných javascriptovou komponentou do xml súboru.

XML -> Json

Transformacia Xml do Json sa používa pri úprave životopisu. Xml je načítané z disku, transformované do Jsonu a odoslané javascriptovej komponente AngularJS, ktorá data predvyplní uživatelovi, rovnako ako keď ich vypĺňal prvý raz.

XML Schema

Xml Schéma nám zabezpečuje validné xml data, konzistenciu dat. Xml schéma kontroluje elementy, atribúty, hodnoty v xml.
